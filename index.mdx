## What Valstore does

Valstore connects to the official Riot Games authentication system and the Valorant PVP API to fetch your personalized store data in real time. No game client required.

## Quick start

<Steps>
  <Step title="Install Flutter">
    Ensure you have Flutter SDK 3.4.1 or higher and Android SDK installed on your development machine.
  </Step>
  <Step title="Clone and set up">
    ```bash
    git clone https://github.com/ruzbyte/valstore-mobile.git
    cd valstore-mobile
    flutter pub get
    ```
  </Step>
  <Step title="Run on device">
    Connect an Android device or start an emulator, then run:
    ```bash
    flutter run
    ```
  </Step>
  <Step title="Log in">
    Open Valstore and log in with your Riot Games account via the embedded browser. Select your region and your shop loads automatically.
  </Step>
</Steps>

<Note>
  Valstore is currently Android-only. iOS support is not planned. The project is archived but fully functional as of the last release.
</Note>
